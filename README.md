**A股量化回测系统 — 基于LSTM的多任务学习策略**

📖 项目简介

本项目构建了一个完整的A股量化回测系统，核心是一个基于 LSTM 的多任务学习模型，同时预测股票次日收盘价（回归任务）和涨跌方向（分类任务）。



系统实现了 “数据获取 → 模型训练 → 策略回测 → 批量筛选 → 结果分析” 的完整闭环，并在全市场 5000+ 只股票上进行了验证，能够自动筛选出策略表现优异的“白名单”股票。



**🎯 项目亮点**

✅ 多任务学习架构：共享 LSTM 底层，同时优化回归和分类任务，提升泛化能力



✅ 工程化策略设计：结合止盈止损、动态阈值，实现“截断亏损，让利润奔跑”



✅ 全市场批量验证：支持并行回测 5000+ 只A股，自动生成策略白名单



✅ 工业级工程优化：包含数据缓存、增量更新、限流器、并行加速等模块



✅ 完整的分析工具：自动生成收益率分布、回撤分析、相关性检验等可视化报告

**🛠️ 技术栈**

核心框架

PyTorch 2.x — 深度学习框架

akshare — A股数据获取

Pandas / NumPy — 数据处理与计算



模型架构

LSTM — 长短期记忆网络，处理时间序列

多任务学习 — 共享底层，多输出（回归 + 分类）

StandardScaler — 数据标准化



工程优化

ThreadPoolExecutor — 并行加速

数据缓存 + 增量更新 — 减少重复下载

限流器 — 避免触发数据源反爬



**📂 项目结构**

├── stock\_full2.py              # 核心回测引擎（训练 + 回测）

├── batch\_backtest.py # 并行批量回测（含缓存和限流）

├── analyze\_results.py          # 结果深度分析（统计 + 可视化）

├── stock\_data\_cache/           # 数据缓存目录（自动生成）

├── whitelist.csv               # 策略白名单（自动生成）

├── batch\_results.csv           # 批量回测结果（自动生成）

└── README.md                   # 项目说明文档



**🚀 快速开始**

1\. 安装依赖

bash

pip install -r requirements.txt

2\. 单只股票回测

bash

python stock\_final.py

程序会交互式询问股票代码，自动完成训练、回测并生成资金曲线图。



3\. 全市场批量回测

bash

python batch\_backtest.py

程序会：



自动获取全市场股票列表

使用缓存 + 并行加速回测

生成 batch\_results.csv 和 whitelist.csv



4\. 结果分析

bash

python analyze\_results.py



自动生成：

收益率分布直方图

收益 vs 回撤散点图

统计摘要报告（CSV）
