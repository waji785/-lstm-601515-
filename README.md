# A股量化交易策略回测系统

## 项目简介
本项目基于 LSTM 神经网络，对 A 股（以 601515 为例）构建了一个多任务预测模型，同时预测**明日收盘价**（回归任务）和**明日涨跌方向**（分类任务），并结合止盈止损策略进行回测。

## 技术栈
- Python 3.12
- PyTorch（模型构建与训练）
- akshare（数据获取）
- pandas / numpy（数据处理）
- matplotlib（结果可视化）

## 核心策略逻辑
1. 使用过去 20 个交易日的量价数据和技术指标（MA5、MA20、波动率等）作为输入。
2. LSTM 模型输出明日上涨概率和预测价格。
3. 当上涨概率 > 买入阈值时开仓，< 卖出阈值时平仓。
4. 引入止盈（+30%）和止损（-10%）规则控制风险。

## 回测表现（2020-2026年）
- 总收益率：**60.79%**
- 交易次数：6 次
- 盈利次数：3 次（平均盈利 +34%）
- 亏损次数：3 次（平均亏损 -12.5%）
- 最大单笔盈利：+35.27%

## 如何运行
1. 安装依赖：`pip install -r requirements.txt`
2. 运行主脚本：`python stock_final.py`
