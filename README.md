# \# 📈 A股量化回测系统 v2.0

# 

# 基于 LSTM 的全市场 A 股量化回测系统。支持单只股票回测、全市场批量回测、自动生成策略友好型股票池（白名单）和深度分析报告。

# 

# \---

# 

# \## 🎯 项目亮点

# 

# \- \*\*深度学习建模\*\*：使用 LSTM 进行多任务学习（同时预测价格和涨跌方向），并整合了止盈止损机制。

# \- \*\*全市场批量回测\*\*：自动遍历全 A 股（5000+ 只），支持并行计算与 GPU 加速，大幅提升回测效率。

# \- \*\*智能股票筛选\*\*：基于回测结果自动生成“白名单”，聚焦策略最擅长的标的。

# \- \*\*深度分析模块\*\*：自动生成收益分布、风险指标、相关性检验和可视化报告。

# \## 🚀 快速开始

# 

# \### 1. 安装依赖

# ```bash

# pip install -r requirements.txt

# 2\. 单只股票回测

# bash

# python stock\_full2.py

# 程序会交互式询问股票代码，自动训练并回测。

# 

# 3\. 全市场批量回测

# bash

# python batch\_backtest.py

# 支持两种模式：

# 

# 全量回测：遍历全市场，生成白名单

# 

# 仅跑白名单：快速验证已有白名单股票

# 

# 4\. 分析回测结果

# bash

# python analyze\_results.py

# 自动生成统计摘要、图表和白名单。

# 

# 📊 输出说明

# 输出	说明

# batch\_results.csv	全市场回测明细结果

# whitelist.csv	策略白名单（高收益 + 低回撤 + 有效交易）

# analysis\_output/\*.png	收益分布、回撤、交易次数可视化

# analysis\_output/\*.csv	统计摘要与描述性统计

# 🧠 策略逻辑

# 输入特征：过去 20 个交易日的量价数据 + 技术指标（MA5、MA20、波动率等）

# 

# 模型架构：LSTM（多任务学习，同时预测价格和涨跌方向）

# 

# 交易规则：

# 

# 上涨概率 > 买入阈值 → 开仓

# 

# 上涨概率 < 卖出阈值 → 平仓

# 

# 止盈（+30%）/ 止损（-10%）强制退出

# 

# 训练轮数：50 轮（平衡速度与精度）
